Generador profesional de prompts para contenidos visuales en redes sociales, desarrollado en HTML, CSS y JavaScript. Esta herramienta permite crear prompts claros, personalizables y listos para usar en generadores de imágenes con IA, facilitando la creación de publicaciones visuales para Instagram, LinkedIn, Facebook y otras plataformas.

El proyecto incluye una interfaz moderna y profesional donde el usuario puede definir el tipo de contenido, formato visual, nivel de complejidad, enfoque, público objetivo y plataforma. Además, incorpora campos opcionales para añadir textos personalizados como titular, subtítulo, llamada a la acción o firma de marca, así como la posibilidad de usar una imagen como referencia visual o fondo.

Su objetivo es agilizar la creación de prompts más precisos, coherentes y útiles para profesionales, docentes, emprendedores y creadores de contenido que quieren generar piezas visuales con mejor dirección creativa y un acabado más cuidado.
